📖 ZARA Language Documentation

Complete reference for the ZARA programming language and its web interpreter.

Language Overview

ZARA is a simple, educational programming language designed to teach fundamental programming concepts. It supports variables, arithmetic operations, string manipulation, conditionals, and loops.

Syntax Reference

1. Variables

Variables are declared using the set keyword.

set variable_name = value
set x = 10
set name = "Hello"
set result = 3.14

2. Output

Use the show keyword to display values.

show 42
show x + 5
show "Hello World"
show name

3. Arithmetic Operations

ZARA supports basic arithmetic operations:

Operator Description Example
+ Addition set result = 5 + 3
- Subtraction set result = 10 - 2
* Multiplication set result = 4 * 6
/ Division set result = 20 / 4

4. String Operations

Strings are enclosed in double quotes. You can concatenate them with the + operator.

set msg = "Hello"
set name = "World"
show msg + " " + name

5. Comparison Operators

Operator Description Example
== Equal to when x == 5: show "Equal"
> Greater than when x > 5: show "Greater"
< Less than when x < 5: show "Less"

6. Conditionals

Execute code conditionally using the when keyword.

when condition: statement

Example:
set age = 20
when age >= 18: show "You are an adult"

7. Loops

Repeat code using the loop keyword.

loop count: statements

Example:
loop 5: show "ZARA"

Example Programs

Arithmetic

set a = 10
set b = 5
set sum = a + b
set product = a * b
show "Sum: " + sum
show "Product: " + product

Conditional Logic

set score = 85
when score >= 90: show "A Grade"
when score >= 80: show "B Grade"
when score >= 70: show "C Grade"

Loops

loop 3: show "Hello!"
set counter = 1
loop 5: show counter

String Operations

set firstName = "John"
set lastName = "Doe"
set fullName = firstName + " " + lastName
show fullName

Quick Tips

  • Variable names are case-sensitive
  • Use double quotes for strings
  • Numbers can be integers or decimals
  • The + operator works for both arithmetic and string concatenation
  • All numbers are treated as floating-point internally

Troubleshooting

Syntax Error: Check your spelling and ensure proper syntax according to the language rules.

Variable not defined: Make sure you've declared the variable with set before using it.

Type mismatch: Ensure you're using appropriate operations for your data types.
